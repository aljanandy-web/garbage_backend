# 🚮 Smart Waste Management System - Backend

Ito ang backend service para sa aming Smart Waste Management System. Binuo ito gamit ang **FastAPI** framework at sumusunod sa **MVC (Model-View-Controller)** architecture.

## 🛠️ Technology Stack
- **Framework:** FastAPI
- **Database:** SQLite (Relational Database)
- **ORM:** SQLAlchemy
- **Data Validation:** Pydantic
- **Server:** Uvicorn
